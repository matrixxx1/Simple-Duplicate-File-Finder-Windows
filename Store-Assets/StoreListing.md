# Simple Duplicate File Finder

## Short description

Find duplicate files quickly by size, name, and hash.

## Full description

Simple Duplicate File Finder is a Windows utility designed around a simple, local-first workflow. The initial app gives users a clear starting surface for pick folders, scan file groups, review likely duplicates, and prepare safe cleanup notes.

## Feature bullets

- Folder-based duplicate scan queue
- Size and hash comparison workflow
- Keep/delete review list
- Local-only cleanup planning
