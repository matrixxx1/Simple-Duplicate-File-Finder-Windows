# Simple Duplicate File Finder
Reserved name is pre-approved as: **Simple Duplicate File Finder**

## Short description

Find duplicate files quickly by size, name, and hash.

## Full description

Simple Duplicate File Finder is a full-featured local-only Windows utility that scans selected folders, compares files by size and SHA-256, highlights duplicate groups, and helps you safely clean them.

## Feature bullets

- Select multiple folders for one scan.
- Filter by minimum size and extension.
- Detect duplicates with size bucketing and SHA-256 hash verification.
- Review each duplicate set and mark files safely.
- Keep newest/oldest, delete selected, or move to quarantine.
- Export full CSV duplicate report.
- Built-in activity logs and About/help panel.
- 15-day fully functional trial and licensing toggle.
